# BE-Test-01

## Pre-requisites:

- Node.js installed
- npm installed

## Structure:

- `input` - source files
- `output` - sorted files

<b>Note:</b> After sorting, the files will be moved from the input directory to the output directory with the corresponding subdirectories.

## Instructions:

```bash
# run sorter
$ npm run sort

# reset files
$ npm run reset

# generate more files
$ npm run generate
```
